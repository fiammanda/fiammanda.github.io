官方文档似乎下线了，不过之前存的用法还能继续。

<!-- more -->

随机：

```
https://source.unsplash.com/random
```

指定尺寸：

```
https://source.unsplash.com/<width>x<height>
```

指定关键词：

```
https://source.unsplash.com/featured/?<keywords>
```

指定尺寸与关键词时可省略：

```
https://source.unsplash.com/<width>x<height>/?<keywords>
```
